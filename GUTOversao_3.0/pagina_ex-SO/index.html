Explicação do sistema operacional

Como é a Iniciação do Sistema Operacional (SO)

A BIOS assim que termina o processo de POST, requisita a inicialização do sistema operacional que está armazenado no Disco Rígido (Hard Disk em inglês que dá a sigla de HD) que armazena dados incluindo o sistema operacional que precisa ser copiado como outros arquivos para a memória RAM (Random Acess Memory) que é responsável por ler todos os dados que passam pelo computador.

O SO depois de ser carregado libera acesso para o usuario fazer login e usar o computador através dos dispositivos de entrada e saída como o mouse e o teclado.

O que faz o sistema operacional:

O SO é responsável por gerenciar os hardwares do computador e controlar qual software vai utilizar uma certa porcentagem do processador, além disso ele também impede que os programas interfiram sobre os outros, então o sistema operacional atua como um gerenciador de recuros sendo uma ponte de ligação entra as entradas feitas pelo usuario e os hardwares.
